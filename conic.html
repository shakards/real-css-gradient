The Conic Gradient

The conic gradient is a type of CSS gradient that creates a smooth transition of
colors around a central point, much like the slices of a pie chart or a color wheel.
Unlike linear or radial gradients, conic gradients transition colors based on angular
direction, radiating around a central point.

This is a basic syntax for conic gradient:

background: conic-gradient(color1, color2, color3, ...);

Result:

In this example, the conic gradient transitions from red to yellow to blue in a circular manner.


  • Starting Angle:

    • The starting angle of a conic gradient determines the point (in degrees) at
    which the first color begins. By default, the gradient starts at 0 degrees
    (the top of the element), but this can be changed.

    You can control the starting point of the gradient using the from keyword, which allows the
    gradient to begin at any angle, measured in degrees. This is particularly useful for creating
    specific effects, such as rotating the gradient.

    Syntax

    background: conic-gradient(from angle, color1, color2, ...);

    CODE

    background: conic-gradient(from 45deg, red, yellow, blue);

    Result:

    In this example, the gradient starts from 45 degrees, creating a rotated effect for the color transitions.


  • Conic Gradient with Unequal Sectors:

    • A conic gradient with unequal sectors is a gradient where each color occupies a different portion of the
    circle, creating a pie-chart-like effect.

    In a standard conic gradient, each color sector is equal in size. However, by specifying angular stops for
    each color, you can control the size of each sector and create custom patterns where some colors occupy
    more space than others.

    Syntax

    background: conic-gradient(color1 start1deg stop1deg, color2 start2deg stop2deg, ...);

    CODE

    background: conic-gradient(red 0deg 120deg, yellow 120deg 240deg, blue 240deg 360deg);

    Result:

    In this example, the gradient has three unequal sectors:

  • Red occupies 0 to 120 degrees,

  • Yellow occupies 120 to 240 degrees,

  • Blue occupies 240 to 360 degrees.


  • Full-Spectrum Color Wheel:

    • A full-spectrum color wheel is a conic gradient that includes a smooth
    transition between all primary and secondary colors, creating a circular rainbow effect.

    This type of conic gradient is often used for creating color wheels, as it transitions
    through the full color spectrum. The gradient is continuous, ensuring that the colors
    blend smoothly across the wheel.

    Syntax

    background: conic-gradient(color1, color2, color3, ..., color1);

    CODE

    background: conic-gradient( red, orange, yellow, green, blue, violet, red);

    Result:

    In this example, the conic gradient smoothly transitions between all the
    rainbow colors and loops back to red, creating a full circular color wheel.


  • Conic Gradient with Transparency

    • A conic gradient with transparency includes transparent or semi-
    transparent colors, allowing part of the background to show through the gradient.

    By using the transparent keyword or RGBA values with alpha transparency, you can
    create gradients where colors fade out, letting the underlying background become visible.
    This technique is useful for creating overlays and layering effects.

    Syntax

    background: conic-gradient(from angle, transparent, color1, color2, transparent);

    CODE

    background: conic-gradient(from 0deg, transparent, red 90deg, yellow 180deg,
    blue 270deg, transparent 360deg);

    Result:

    In this example, the gradient transitions from transparent to red, yellow, and
    blue, and back to transparent, revealing parts of the elementâ€™s background.


    Repeating Conic Gradients

    A repeating conic gradient repeats a set of colors in a loop
    around the center of the element, creating a continuous, patterned effect.

    Repeating conic gradients are useful for creating striped or circular patterns
    that repeat infinitely within the element.
    The gradient repeats itself once it completes the full 360 degrees.

    Syntax

    background: repeating-conic-gradient(color1 stop1deg, color2 stop2deg, ...);

    CODE

    background: repeating-conic-gradient(red 0deg, yellow 30deg, blue 60deg);

    Result:

    In this example, the gradient repeats every 60 degrees, creating a
    circular striped pattern with alternating red, yellow, and blue segments.

    Conclution

    Conic gradients in CSS offer a powerful way to create visually dynamic
    and complex color patterns. By manipulating the starting angle, sector
    sizes, transparency, and repeating patterns, designers can achieve a
    wide range of effects suitable for everything from simple backgrounds
    to intricate visual elements.
